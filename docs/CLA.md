# TARS 贡献者许可协议（CLA）简版草案

本协议适用于向 TARS 项目提交贡献的个人或组织（下称“贡献者”）。提交代码、文档或其他材料即视为接受本协议。

## 1. 贡献的定义
“贡献”指贡献者提交或以其他方式提供并被 TARS 项目接收的任何代码、文档、设计、测试用例或其他材料。

## 2. 许可授予
贡献者授予 TARS 项目维护方及其用户一项全球范围内、永久、不可撤销、免版税、可再许可的版权许可，用于使用、复制、修改、分发、公开展示及以任何形式再许可其贡献。该许可应至少与 TARS 项目当前采用的开源许可证相兼容。

## 3. 专利许可
贡献者授予 TARS 项目维护方及其用户一项全球范围内、永久、不可撤销、免版税的专利许可，允许其在贡献所必需的范围内使用、制造、销售、提供、进口及以其他方式实施贡献。

## 4. 贡献者声明
贡献者声明并保证：
- 其拥有合法权利提交贡献，并授予本协议所述许可；
- 贡献不侵犯任何第三方的知识产权或其他权利；
- 如贡献包含第三方材料，已获得必要许可并在提交时明确标注。

## 5. 不保证
贡献以“现状”提供，维护方不提供任何明示或默示担保。

## 6. 适用法律
本协议的适用法律和争议解决方式由维护方所在地法律确定，除非双方另有约定。
